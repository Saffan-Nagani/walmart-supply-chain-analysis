

## Overview
This project analyzes 6,435 weeks of Walmart sales data across 45 stores 
using SQL to identify demand patterns and provide actionable inventory 
planning recommendations.

## Business Problem
Without understanding which stores drive the most revenue, how holidays 
impact demand, and how unemployment affects sales, inventory planning 
becomes reactive this can lead to stockouts at high-demand stores and 
overstock at low-demand ones.

## Tools Used
- SQL (DB Browser for SQLite)
- Microsoft Word / PDF for reporting

## Dataset
Walmart Weekly Sales Dataset — sourced from Kaggle
6,435 records | 45 stores | 8 variables

## Key Findings
- Store 20 generates 8x more revenue than the lowest performer (Store 33)
- Holiday weeks drive 8% higher average sales across all stores
- Store 10 shows the highest holiday sensitivity i.e., $230K spike per holiday week
- Stores in low unemployment areas generate 38% more sales than high unemployment areas
